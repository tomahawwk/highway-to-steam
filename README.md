# Test task for Finom

A modal window with a banner.

[Live demo](https://finom-task.vercel.app/)

Technical stack: React, Vite, Typescript, Tailwind.

The modal window is designed as a separate universal component with the following functionality:

- closing with a button with a cross
- closing on ESC
- click closure outside the modal window area
- removal from DOM after closing

It can be reused with any content.

## Project structure:

_/global-styles/theme.css_ - css variables  
_/components_ - application components  
_/icons_ - icons
_/ui_ - interface elements
